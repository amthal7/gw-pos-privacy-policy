Privacy Policy

Last updated: March 18, 2025

1. Introduction

Welcome to GW Gateway. This Privacy Policy explains how we handle data, permissions, and security measures within the app.

GW Gateway is a merchant-focused application that allows authorized users to purchase telecom recharge e-vouchers using an internal wallet system. The app does not support user sign-up, external wallet top-ups, or third-party transactions.

2. Information We Collect

GW Gateway does not collect, store, or share any personal data. The app operates within a closed merchant network and does not require user registration.

However, the app may process certain data to ensure proper functionality:

  • Transaction History: Logs in-app purchases for internal accounting and reporting.
  • Device Information: May collect basic device identifiers to prevent unauthorized access and ensure security.

3. App Permissions

GW Gateway requires the following device permissions:

  • Location: Used to verify merchant location for security and fraud prevention.
  • Bluetooth: Used to connect with Bluetooth printers for printing transaction receipts.
  • Notifications: Used to send transaction alerts and important updates.

These permissions are used solely for operational purposes and are not shared with third parties.

4. How We Use Data

The data processed within GW Gateway is used strictly for operational purposes:

  • Facilitating secure transactions within the internal merchant network.
  • Ensuring fraud prevention and security compliance.
  • Improving the reliability and performance of the app.

5. Data Retention & Security

GW Gateway does not permanently store any sensitive user data. Transaction logs are retained for internal accounting and fraud prevention, following strict security protocols.

All data is processed within a secure environment and is never shared with third-party entities.

6. User Responsibilities

Authorized merchants using the app are responsible for:

  • Keeping their login credentials secure and confidential.
  • Ensuring transactions are performed in a secure manner.
  • Reporting any suspicious activity to the company.

7. Third-Party Access

The app does not integrate with third-party services for data collection, analytics, or advertising. All transactions occur within the internal merchant network.

8. Compliance with Laws

GW Gateway complies with applicable data protection and privacy laws. Since the app does not collect personal information, it does not fall under consumer data protection regulations such as GDPR or CCPA.

9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page, and continued use of the app after updates means acceptance of the new terms.
